# Description
FullStack project. 
* Frontend - React, GraphQL, Apollo. CRUD operations, queries, refetching, polling, mutations, subscriptions, error handling by Apollo Link, authentication by access token.
* Backend: Node.js, Express, GraphQL, Apollo.
